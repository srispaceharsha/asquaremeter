Colophon

This page celebrates the tools and technologies that power One Square Meter.

Source Code

This project is open source. The code is split into two repositories:

  • asquaremeter - The static site (HTML, CSS, templates)
  • asquaremeter-cms - The pipeline and build tools

Built With

Backend & Pipeline

  • Python - The language powering the data pipeline and site generator
  • Jinja2 - Templating engine for generating HTML pages
  • Pillow - Image processing for thumbnails and web-sized images
  • PyEphem - Calculating moon phases and sun times
  • Open-Meteo - Weather data API
  • GBIF - Taxonomy data for species identification
  • Boto3 - Uploading images to cloud storage

Frontend

  • Vanilla HTML, CSS & JavaScript - No frameworks, just the basics
  • Google Fonts - Typography

Infrastructure

  • Cloudflare Pages - Static site hosting
  • Cloudflare R2 - Image storage and CDN
  • GitHub - Version control and CI/CD

Code Authorship

The codebase for this project was written collaboratively with Claude, an AI assistant by Anthropic. From the data pipeline to the static site generator, from image processing to taxonomy lookups - Claude helped architect and implement it all through conversation and iteration.

A human provided the vision, the domain knowledge, and the bugs to document. An AI provided the code. Together, we built something neither could have alone.
